# Personal & Technical Companion - Frontend

React frontend for the Personal & Technical Companion application.

## Features
- Task management with priorities
- Technical Q&A with NLP capabilities
- Goal tracking with progress visualization
- Modular component architecture

## Tech Stack
- React 18
- Axios for API calls
- Framer Motion for animations
- React Router for navigation

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation
```bash
npm install
npm start
